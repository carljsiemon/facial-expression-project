Finding the challenging problem of image classification both fascinating and rewarding I decided upon the task of facial expression classification for my capstone project during the winter 2015 Zipfian Academy Data Science Cohort at Galvanize.

The data I used can be found here. In order to make the project feasible in the allotted time I reduced the set down to three classes (Happy, Sad, and Surprised), which still left approximately 19,000 labeled images to train on. For classification I trained a convolutional neural network using GraphLab Create™ with GPU Acceleration and acheived 75% accuracy. The modeling code can be found on GitHub.

Technologies used for this project:

  • Python
  • GraphLab Create™ with GPU Acceleration
  • OpenCV
  • Numpy, Pandas
  • AWS EC2 GPU Instance
  • Django (web)
